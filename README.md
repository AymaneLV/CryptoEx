## Getting Started

### Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)
  
### Install and Run
1. Clone the repo:
    ```bash
    git clone https://github.com/AymaneLV/CryptoEx.git
    cd CryptoEx
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the app:
    ```bash
    npm start
    ```
